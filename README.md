# 🎓 Student Fitness AI

A web-based fitness platform designed for students to track and improve their health using AI-based suggestions.

---

## 🚀 Features

- Student fitness tracking
- Workout suggestions
- Clean and responsive UI
- Simple and easy to use

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

Student-Fitness
│
├── backend
├── css

└── rough.html

---

## 💻 How to Run

1. Download the project
2. Open the folder
3. Run `index.html` in your browser
